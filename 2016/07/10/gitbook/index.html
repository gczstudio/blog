怎么将gitbook导出为html格式

发表于 2016-07-10   |   分类于 gitbook   |  

最近使用了gitbook来记录笔记,发现挺好用,使用markdown语法书写,然后可以转为html页面,查看起来很方便.

如何使用?

  1. 安装node.js环境,可以去node.js官网进行下载
    事先查看自己电脑是32位还是64位

    • 方法一:右键点击我的电脑–>查看属性
    • 方法二:win+R,输出cmd,进入命令行,输入systeminfo,查看系统类型

  2. node.js安装完成后,进入命令行
    输入node -v查看node.js的版本号
    然后输入npm -v查看npm的版本号
    能显示出版本号,说明安装成功

  3. 输入npm install -g gitbook-cli

  4. 输入gitbook -V 查看gitbook-cli的版本号,会自动安装Gitbook,耐心等待.

  5. 安装gitbook软件

  6. 打开gitbook软件,需要进行注册,如果你不想注册的话,点击下面的do This Later,就可以直接使用了

  7. 书写好的文件存放在C:\Users\gcz\GitBook\Library\Import下

  8. 导出: 命令行下 cd C:\Users\gcz\GitBook\Library\Import\bookName bookName是你要导出文件的名字
    再输入 gitbook build 就可以导出了,导出成功后,在你导出的文件目录下会有一个_book文件夹,html就存在里面.
