git基础命令

发表于 2016-07-14   |   分类于 git   |  

  1. 配置用户
    git config --global user.name "自已的名字"
    git config --global user.email "自已的邮箱地址"
    注:配置用户只需要执行1次,可以重复使用
    git config --list可以查看设置的用户信息

  2. 初始化仓库

    • git init
      创建了一个名为.git的隐藏目录,这个目录就是存储我们历史版本的仓库,ls -al可以查看
    • git clone 仓库地址
      执行完这个命令，会在当前目录下生成一个Monment目录（默认和仓库名称相同），这个便是已有一个使用Git管理的项目。

  3. 查看文件状态
    git status

  4. 添加文件到暂存区
    git add 文件名/文件夹 "*" 或者-A代表所有

  5. 撤销更改
    git checkout 文件名
    注:从暂存区还原到工作区

  6. 提交文件
    git commit -m '备注信息'
    将暂存区标记为绿色的文件,全部提交到本地仓库存储

  7. 查看提交历史
    git log
    注:按q键退出

  8. 恢复上一次提交的状态
    git reset --hard c888a614e072e2
    c888a614e072e2使用git log查看对应的历史id
