关于Hexo插入图片的问题

发表于 2016-06-10   |   分类于 hexo   |  

原因

在hexo中使用本地图片，利用markdown书写时,![imgName](path/imgName.jpg)使用相对路径时,图片无法在网页上正常显示
但是使用http://…. 路径网络图片时,是可以正常显示的.

解决办法

使用hexo-asset-image

  1. 首先确认_config.yml 中有post_asset_folder: true

  2. 在hexo目录下,执行
    npm install https://github.com/CodeFalling/hexo-asset-image –save

3.保证你的xxx.md文件和存放图片的文件夹名字保持一致,并且在同一目录下,
只要使用![1.png](insertImage/1.jpg),就可以正常插入图片了.
如下图:把1.png插入到insertImage.md中,1.png保存在insertImage文件夹中
